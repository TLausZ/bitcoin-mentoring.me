Bitcoin Node zusammenbauen

Bitcoin Lightning Node zusammenbauen und syncen

Jeder kennt es, man kauft sich Hardware (z.B. einen Raspberry Pi) und versucht nun einen Lightning / Bitcoin Node darauf aufzusetzen. Doch wie funktioniert das genau? Was ist eine SSH Verbindung? Wie kriege ich das Betriebssystem auf den Raspberry Pi? Auf was muss ich beim Einrichten genau achten?

In diesem online Kurs werden wir euch durch den Dschungel der Bitcoin Lightning Fullnodes führen. Folgende Themen werden im Kurs beinhaltet sein:

  • Hardware Zusammenbau
  • Raspiblitz OS auf die SD Karte flashen
  • Einrichtung der Lightning Node
  • Start des Blockdownloads

In diesem Kurs werden wir das Einrichten der Lightning Node mit Raspiblitz zusammen durchführen. Damit gleich mitgemacht werden kann, sollte die Hardware bereits im Vorfeld gekauft worden und bereit für die Installation sein. Dafür haben wir hier eine Einkaufsliste zusammengestellt:

  • Raspberry Pi (min. 4GB-8GB RAM) oder vom Fulmo Shop (min. 4GB RAM)
  • Raspbbery Power Netzteil (Achtung: Auf richtiges Land achten!)
  • Externe SSD Festplatte 1TB
  • Sata zu USB Adapter
  • Samsung SD Karte 32 GB
  • Raspberry Gehäuse (Optional)

Lernziele

Nach diesem Kurs verstehst du wie du deinen Bitcoin / Lightning Node einrichten kannst. Die Blockchain wird gesyncht und der Lightning Node ist eingerichtet

Zielgruppe

Technik interessierte, die gerne selber zu Hause einen Node betreiben wollen.

Voraussetzung

  • Hardware wurde gekauft und liegt zur Installation bereit.

Veranstaltungsort

Dieser Kurs wird online über die Opensource Plattform Jitsi durchgeführt. Der Zugangslink erhalten Sie nach der Anmeldung.
